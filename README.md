# GlamWigs E-commerce Website

GlamWigs is a modern e-commerce platform built with Next.js, offering a seamless shopping experience for premium wigs and hair products. The website features a responsive design, integrated WhatsApp communication, and an intuitive shopping cart system.

## Key Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Product Catalog**: Organized display of wig collections with detailed descriptions
- **Shopping Cart**: Easy-to-use cart functionality for managing purchases
- **WhatsApp Integration**: Direct communication with customer service
- **SEO Optimized**: Enhanced for better search engine visibility
- **Mobile-First Approach**: Smooth mobile shopping experience

## Tech Stack

- [Next.js](https://nextjs.org/) - React framework for production
- TailwindCSS - For responsive styling
- TypeScript - For type-safe code
- Shadcn UI - For consistent UI components

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
/app          - Next.js app directory containing pages and routes
/components   - Reusable UI components
/lib          - Utility functions and data models
/public       - Static assets like images
```
